**Setup:**
1. Maak een kanaal genaamd `#counting`, of stuur `c!channel` in het kanaal waarin je wilt tellen.
2. Start met tellen in dat kanaal.

**Gebruiker Commando's:** *Waar `?` staat, is iets invullen optioneel.*
• `c!cs ?[paginanummer]` - Bekijk de huidige top tien van serverscores.
• `c!lb ?[paginanummer]` - Bekijk de top tien server high scores.
• `c!ulb ?[paginanummer]` - Bekijk de top tien gebruikers.
• `c!slb ?[paginanummer]` - Bekijk de top tien gebruikers in jouw groep.
• `c!user ?[@user/user id]` - Bekijk de stats van een gebruiker.
• `c!server` - Bekijk de stats van de server waar je in bent.
• `c!donatesave` - Doneer een save aan de server waar je in bent. (voegt `0.2` van een groepssave toe.)
• `c!transfersave [@user/user id]` - Geef een save cadeau aan een andere gebruiker. Dit kost `0.5` save (dus `1.5` in totaal.
• `c!vote` - Stem voor de bot en ontvang saves.
• `c!stats` - Bekijk de statistieken voor de bot en de huidige shard.

**Beheerder Commando's:**
• `c!failrole [@role/role id]` - Bepaal een rol die gebruikers krijgen als ze verkeerd tellen. Stuur dit commando zonder een rol om dit terug te draaien.
• `c!channel` - Maak van het huidige kanaal het telkanaal.
• `c!numbersonly [on/off]` - Schakel de alleen-nummers mode om. Dit zorgt ervoor dat je opnieuw moet beginnen als er een bericht gestuurd wordt dat niet een commando of een correct nummer is.

**Saves:**
• Stem [hier](https://counting.duckgroup.xyz/vote) om meer saves te verdienen. Check hoeveel je hebt met `c!vote`.
• Als je tenminste één save hebt, kan je doorgaan met tellen nadat je een verkeerd nummer gestuurd hebt.

**Tel regels:**
• Je kan niet twee keer achter elkaar tellen.
• Gebruikt geen robots om voor je te tellen.
• Als je verkeerd telt, wordt er een save gebruikt als je die hebt. Als je die niet hebt, moet je opnieuw beginnen.
